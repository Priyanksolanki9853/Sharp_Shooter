## üéÆ About The Game
An intense **3D First-Person Shooter (FPS)** developed using Unity and C#. 

Set in a stylized sci-fi arena, the player must survive against waves of hostile enemy drones and shooters. The game features fast-paced movement, various weapon types, and an objective-based combat system. The goal is simple: eliminate all enemies to clear the level.

## ‚ú® Key Features
* **First-Person Movement:** Smooth FPS controller with jumping and sprinting mechanics.
* **Enemy AI:** Intelligent bot system that tracks and engages the player.
    * *Flying Drones* that swarm the player.
    * *Ground Shooters* with ranged attacks.
* **Weapon System:** Multiple weapon types including:
    * Rapid-fire Blasters.
    * Sniper Rifles for long-range engagements.
* **UI & HUD:** Real-time health/shield monitoring, ammo counters, and an "Enemies Remaining" tracker.
* **Win/Loss Logic:** Complete "Game Over" and "Victory" states with restart functionality.
